# ระบบเช็คชื่อเข้าเรียน | Student Attendance System

ระบบเช็คชื่อเข้าเรียนออนไลน์สำหรับนักเรียนระดับ ปวช. และ ปวส. พร้อมการบันทึกประวัติการเข้าเรียน

## 🎯 คุณสมบัติ

✅ **ฟอร์มกรอกข้อมูล**
- รหัสนักศึกษา
- ชื่อ-นามสกุล
- เลือกระดับชั้น (ปวช./ปวส.)
- เลือกสาขาวิชา (อัปเดตตามระดับชั้นที่เลือก)

✅ **บันทึกประวัติ**
- บันทึกวันที่และเวลา อัตโนมัติ
- แสดงรายการเช็คชื่อในตาราง
- นับจำนวนรายการทั้งหมด

✅ **ออกแบบ**
- Responsive design (ใช้งานได้บนมือถือและเดสก์ท็อป)
- อินเตอร์เฟส ไทย สวยงาม
- Tailwind CSS สำหรับ styling
- Icons จาก Lucide

## 📋 สาขาวิชาที่รองรับ

### ระดับ ปวช. (Vocational Certificate)
- ช่างยนต์
- ช่างกลโรงงาน
- ช่างไฟฟ้ากำลัง
- อิเล็กทรอนิกส์
- ช่างก่อสร้าง
- สถาปัตยกรรม
- การบัญชี
- การตลาด
- คอมพิวเตอร์ธุรกิจ
- เทคโนโลยีสารสนเทศ

### ระดับ ปวส. (Higher Vocational Certificate)
- เทคนิคยานยนต์
- เทคนิคการผลิต
- ไฟฟ้ากำลัง
- อิเล็กทรอนิกส์อุตสาหกรรม
- ช่างโยธา
- เทคนิคสถาปัตยกรรม
- การบัญชี
- การตลาด
- เทคโนโลยีธุรกิจดิจิทัล
- นักพัฒนาซอฟต์แวร์

## 🚀 วิธีการใช้งาน

### ตัวเลือก 1: ใช้งานโดยตรง
1. ดาวน์โหลดไฟล์ `attendance_system.html`
2. เปิดไฟล์ด้วย Web Browser (Chrome, Firefox, Safari, Edge)
3. เริ่มใช้ระบบได้ทันที

### ตัวเลือก 2: อัปโหลด GitHub Pages
1. สร้าง Repository ใหม่ที่ชื่อ `username.github.io`
2. อัปโหลดไฟล์ `attendance_system.html` ไปยัง Repository
3. เข้าถึงได้ที่ `https://username.github.io/attendance_system.html`

### ตัวเลือก 3: อัปโหลด GitHub ทั่วไป
1. สร้าง Repository ใหม่ (เช่น `attendance-system`)
2. อัปโหลดไฟล์ `attendance_system.html`
3. ไปที่ Settings → Pages
4. เลือก "Deploy from a branch"
5. เลือก Branch `main` และ folder `/root`
6. เข้าถึงได้ที่ `https://username.github.io/attendance-system/`

## 💾 การจัดเก็บข้อมูล

- ข้อมูลถูกจัดเก็บใน **Session Storage** (เฉพาะ Browser ปัจจุบัน)
- ข้อมูลจะหายไปเมื่อปิด Browser
- หากต้องการเก็บข้อมูลถาวร สามารถต่อเชื่อมกับ Database ได้

- **Tailwind CSS** - Styling (ผ่าน CDN)
- **Lucide Icons** - Icons (ผ่าน CDN)
- **Vanilla JavaScript** - Logic

## 🔧 การปรับแต่ง

### เพิ่มสาขาวิชา
แก้ไข Array ของ `VOCATIONAL_BRANCHES` หรือ `HIGH_VOCATIONAL_BRANCHES` ในส่วน `<script>`

```javascript
const VOCATIONAL_BRANCHES = [
    "ช่างยนต์",
    "สาขาใหม่", // เพิ่มสาขาใหม่ที่นี่
];
```

### เปลี่ยนสี
ค้นหา `bg-blue-` ใน HTML และเปลี่ยนเป็นสี Tailwind ที่ต้องการ
- `bg-blue-600` → `bg-red-600`
- `text-blue-700` → `text-green-700`

## 📄 ข้อมูลในตาราง

| ฟิลด์ | คำอธิบาย |
|------|---------|
| วันที่ | วันที่ลงทะเบียน (รูปแบบไทย) |
| เวลา | เวลาที่ลงทะเบียน (HH:MM:SS) |
| รหัสนักศึกษา | รหัสประจำตัวนักศึกษา |
| ชื่อ-นามสกุล | ชื่อเต็มของนักศึกษา |
| ระดับ/สาขา | ระดับชั้น และ สาขาวิชา |

## ⚠️ ข้อสำคัญ

- ข้อมูลจัดเก็บเฉพาะใน Browser ของผู้ใช้
- ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์
- ปิด Browser = ข้อมูลหายไป (เว้นแต่บันทึกไปยัง Excel หรือ PDF)

- ไม่มีการส่งข้อมูลบนอินเทอร์เน็ต
- ใช้งานแบบออฟไลน์ได้
